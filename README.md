# Portafolio Web - Juan Felipe Arias

## Descripción del proyecto

Este proyecto consiste en un portafolio web responsive desarrollado para presentar mis habilidades, experiencia y proyectos como desarrollador web junior y productor audiovisual. El sitio incluye una sección principal de presentación, habilidades y herramientas, proyectos destacados y un apartado de contacto con enlaces a redes sociales.

El diseño está inspirado en interfaces modernas tipo portfolio, utilizando efectos visuales, hover animations y responsive design para adaptarse a diferentes resoluciones de pantalla.

---

## Funcionalidad

- Presentación personal y profesional.
- Sección de habilidades y herramientas.
- Tarjetas de proyectos destacados.
- Enlaces interactivos con efectos hover.
- Diseño responsive para desktop, tablet y mobile.
- Enlaces de contacto y redes sociales.
