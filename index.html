Página web de una cafetería

Desarrollé la página web de Triple Expresso desde cero, encargándome tanto del diseño como de la implementación. Utilicé HTML y CSS con Flexbox para lograr una estructura moderna, responsive y visualmente atractiva, alineada con la identidad de la cafetería. El proyecto fue gestionado con GitHub, permitiendo un flujo de trabajo organizado y escalable. El resultado es una experiencia web clara, funcional y enfocada en resaltar la esencia y los productos de la marca.
